# Sorting Methods

This folder contains Python implementations of fundamental sorting algorithms.

The purpose of this folder is to improve algorithmic thinking and understand how sorting works step by step.

## Included Algorithms

- Bubble Sort (`bubble_sort.py`)
- Selection Sort (`selection_sort.py`)
- Insertion Sort (`insertion_sort.py`)

## Complexity

| Algorithm       | File Name           | Best Case | Worst Case |
|----------------|---------------------|-----------|------------|
| Bubble Sort    | `bubble_sort.py`    | O(n)      | O(n²)      |
| Selection Sort | `selection_sort.py` | O(n²)     | O(n²)      |
| Insertion Sort | `insertion_sort.py` | O(n)      | O(n²)      |

## Learning Goals

This folder focuses on:

- comparing elements
- swapping values
- sorting logic
- time complexity basics
- step-by-step algorithm understanding
