# Search Methods

This folder contains Python implementations of searching and graph traversal algorithms.

The goal is to understand how different search techniques work on arrays and graphs.

## Included Algorithms

- Jump Search
- Breadth-First Search (BFS)
- Depth-First Search (DFS)

## Learning Goals

This folder focuses on:

- search efficiency
- graph traversal
- queue and stack usage
- recursion and iterative logic
- path exploration

## Notes

These implementations are written for learning and practice purposes.
